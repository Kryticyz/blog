Building web applications doesn't have to be complicated.

## Keep It Simple

The best projects start with minimal dependencies and clear architecture.

### Benefits

- Easier to maintain
- Faster to develop
- Less technical debt

## Code Example

```go
func main() {
    log.Println("Simple is better")
}
```

That's it!
